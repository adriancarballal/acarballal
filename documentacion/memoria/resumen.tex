\section*{Resumen}
El proyecto \textit{El mas} es un concurso-reality multiplataforma (televisión, Web, móvil), en el que los concursantes envían un video de hasta 90 segundos cada semana. Esos videos son votados mediante una plataforma Web y los que obtienen más votos son emitidos en un programa de televisión semanal de 30 minutos de duración. 


En el presente proyecto se acomete el diseño e implementación de un portal web destinado a albergar el contenido multimedia de un espacio televisivo. A su vez éste debe servir como fuente de contenidos. Para facilitar la difusión del programa se realizará una aplicación web que permita su manejo tanto en navegadores convencionales como en navegadores utilizados por dispositivos móviles.


La aplicación resultante de este proyecto conformará parte de un proyecto conjunto entre la Facultad de Informática y la Facultad de Ciencias de la Comunicación de la Universidad de la Coruña. De este modo, el sistema tendrá que ser probado adecuadamente, ya que se presume su posible comercia\-lización.


La aplicación desarrollada permite automatizar labores de administración (comprobación de videos, recodificación, gestión de abusos, elaboración de listas de videos más votados para elaborar el programa de tv) y da soporte completo a la votación e incorporación de videos. También permite una serie de utilidades como realizar comentarios, notificar abusos, etc. La aplicación emplea tecnologías libres y avanzadas como Hibernate, Spring y Tapestry.
Se ha implementará un API completo que sirva de enlace entre la aplicación
web y un sistema de compresión de video instalado en el propio servidor de
la aplicación.


Además de una descripción de los detalles de implementación, el proceso de desarrollo, la metodología seguida y las características finales del producto, se exponen en esta memoria los resultados de las pruebas de unidad, integración y estrés. Se proporcionan los manuales necesarios para instalar y adaptar el producto.
